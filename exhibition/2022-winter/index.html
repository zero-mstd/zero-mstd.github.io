生活

今天城市探索，看到一条铁路，于是顺着铁轨走了一个多小时. 这条铁路穿过一片树林，景色优美，空气清新，阳光斑驳，时间都安静了. 铁路不笔直，目之所及有限，很激发人的探索欲. 我四五岁的时候住的地方附近有一条铁轨，那时候我常常去玩，长大后很少有机会走铁轨了~

我本来以为这条铁路已经废弃了，结果我走着走着前方突然传来火车的鸣声，我赶紧跳到旁边去……过了一小会儿还真的开来一列火车，货运的，可能是因为这地方是弯道吧，火车开得很慢. 数了一下，一共只有十节车厢，不到一分钟就从我旁边开过去了. 我脑海中闪过一个念头，那就是我很想扒上这列火车，然后任凭它带我去往任何地方！但我观察了一下，后面的车厢其实很难有地方可以稳稳扒住，还有就是虽然火车开得很慢，但如果想上车的话，还是太快了，很有难度 :0120: 这个时候我想起游戏 Life is strange: before the storm 里面的情节，两个主角从学校里逃课出来，就扒上一列火车去了一个很远的地方……我就没做过这种疯狂的事情，我的人生实在是过于波澜不惊了一点 :0120:

原来这件事是从去年年底才刚刚开始做，做得非常不完善，流程必须是手机厂商去申请，他们经济与信息化局才能审批，然后才能通过认定，然后才会把名单给到北京市商务局，然后北京市商务局把名单给京东.

京东去年只支持 18 台手机，是因为确确实实只通过了这些，很多手机厂商不知道，没有申请，他们会找人家手机厂商谈，但不能强迫手机厂商参与，说今年应该能慢慢地支持更多. 说我们消费者能做的可以是去催一催手机厂商，让他们手机厂商主动去申请，他们经济和信息化局不会卡流程，基本都会通过的.

又问这个列表有没有在什么地方公示呢？答曰，没有，列表是经济和信息化局和商务局内部的文件，市民就去京东上看就行，京东上支持什么，就有什么.

技术

之前有象友推荐了一个健康码项目（https://alive.bar/@boop/109265900174401155 ），它给出了两个需要翻墙的在线地址. 这只是一个前端项目而已，有没有可能不联网，在本地运行呢？其实只要下载源码，并运行一个 HTTP server 就可以了. 具体的操作步骤如下（android 手机为例）：

1. 下载项目源码（https://github.com/ilovexjp/health-code-index/releases ），将
health-code-index.zip 解压到手机上；
2. 下载安装一个 HTTP server 服务，比如 ServDroid（https://f-droid.org/en/packages/org.servDroid.web/ ），仅 0.9 Mb；
3. 打开 ServDroid，右上三个点，Settings，Document Path，改成步骤 1 那个文件夹；
4. 回到 ServDroid 首页，点击那个 Off 将 server 打开（再点一下可以关闭）；
5. 打开手机浏览器，输入 0.0.0.0:8080 即可正常使用；
6. 可以添加到主屏幕（pwa 功能）以全屏.

其他系统如 ios 思路一样，找一个 HTTP server 就好了~

补充：

根据象友反馈，如果连着 WiFi，软件上会给出一个局域网的地址，以 192.168 开头，那样的话就在浏览器里输入这个局域网地址就好~
